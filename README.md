# Scratch教学平台功能清单

## 一、用户系统

### 1.1 用户管理
- 用户创建（仅管理员创建）
- 用户登录（账号密码）
- 用户信息管理（个人资料、学习偏好）
- 角色管理（学生、教师、管理员）
- 账号状态管理（启用/禁用）

### 1.2 权限控制
- RBAC权限模型
- 课程访问权限
- 功能模块权限
- 数据隔离（按tenant_id）

## 二、备课系统

### 2.1 课程管理
- 课程大纲管理（树形结构）
- 课程分类与标签
- 课程版本管理
- 课程状态管理（草稿/审核中/已发布/已下线）

### 2.2 教学资源库
- Scratch项目文件上传（.sb3）
- Scratch角色文件上传（.sprite2）
- Scratch背景文件上传（.svg, .png）
- Scratch音频文件上传（.mp3, .wav）
- 示例代码库
- 教学素材库（图片、音频、动画）
- 文件预览（.sb3文件在线预览）
- 文件引用计数

### 2.3 教案编辑器
- 富文本编辑（文字、图片、视频）
- 代码片段插入与高亮
- 互动环节设计（提问、练习）
- 教学目标设定

### 2.4 作业模板库
- 作业类型模板（编程题、选择题、填空题）
- 作业难度分级
- 作业计时设置
- 作业提交要求配置

## 三、授课系统

### 3.1 实时课堂
- 互动白板（代码演示、绘图）
- 屏幕共享
- 课堂录制与回放（本地录制）

### 3.2 学生状态监控
- 在线状态显示
- 代码进度跟踪
- 课堂参与度统计

### 3.3 课堂互动
- 举手提问
- 实时问答
- 投票调查
- 小组讨论

### 3.4 课堂点评功能
- 点评维度（积极、专注、创新、合作等）
- 实时点评统计
- 点评历史记录
- 点评数据可视化（参考班级优化大师）

### 3.5 课堂笔记
- 语雀文档集成
- 笔记自动创建与关联
- 学生可编辑/教师可审核
- 笔记历史版本管理
- 内容解析（通过URL解析课程内容）

## 四、练习系统

### 4.1 在线IDE
- 代码编辑器（支持Scratch）
- 语法高亮与智能提示
- 代码模板系统
- 多语言支持（后续支持Python/C++）

### 4.2 代码执行
- Scratch VM沙箱环境
- 资源限制（CPU/内存）
- 安全隔离
- 执行超时控制

### 4.3 实时反馈
- 编译错误提示
- 运行时错误诊断
- 代码质量建议
- 性能优化建议

### 4.4 代码版本管理
- 代码历史记录
- 版本对比
- 代码回滚
- 代码导出

## 五、自动化测评系统
